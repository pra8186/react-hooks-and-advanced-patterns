# Frontend (Vite + React + TypeScript)

This package contains the tax workspace UI. **Project overview, setup, API notes, and specifications** are documented in the repository root:

[../README.md](../README.md)

## Quick commands (from `frontend/`)

```bash
npm install
npm run dev
npm run build
npm run lint
```

From the monorepo root you can run `npm run dev` (see root `package.json`), which delegates here.
